# Turtle Crossing

A simple frogger-style game where you guide your turtle across a busy road without hitting cars.

## How to Play

- **Up Arrow** — Move forward
- Reach the top of the screen to advance to the next level
- Avoid the cars or game over!
- Each level the cars get faster

## Requirements

- Python 3.x
- `turtle` (built-in with Python)

## Game Files

- `main.py` — Main game loop
- `player.py` — Player turtle
- `car_manager.py` — Creates and moves cars
- `scoreboard.py` — Displays level and game over message
